# Inventory Management System in C

A console-based inventory management application written in C.

## Features

- Add Product
- View Products
- Search Product
- Update Product
- Delete Product
- Sell Product
- Sales Report
- Inventory Value Report
- File Persistence

## Concepts Used

- Structures
- Arrays
- Functions
- CRUD Operations
- File Handling
- Inventory Tracking

## Compile

```bash
gcc main.c -o inventory
./inventory
